1. Subatomic Particle Relationships

Overview

(These ALWAYS appear in KAAC clues.)

  • Protons, Neutrons, Electrons
  • Atomic Number (Z) = number of protons
  • Mass Number (A) = protons + neutrons
  • Neutrons = A − Z
  • Electrons (neutral atom) = protons
  • Electrons (ion) = protons ∓ charge (e.g., Fe3+ has 3 fewer electrons; Cl− has 1 more electron)

Pattern: Mass number changes → neutrons change; Charge changes → electrons change; Atomic number NEVER changes (unless nuclear).

Practice Questions

  1. 1. What subatomic particle determines an element’s identity and equals the atomic number?

    Proton

  2. 2. If an atom has mass number 23 and atomic number 11, how many neutrons does it contain?

    12

  3. 3. What is the charge on an ion that has 10 protons and 8 electrons?

    +2

  4. 4. Changing which subatomic particle creates an isotope?

    Neutrons

  5. 5. A neutral atom becomes larger when it gains electrons. What type of ion is formed?

    Anion

Early-Buzz Patterns

  • “This particle determines the identity…” → Proton
  • “This subatomic particle changes when ions form…” → Electrons
  • “This value equals protons plus neutrons…” → Mass number
  • “This particle varies among isotopes…” → Neutrons
  • “This equals protons in a neutral atom…” → Electrons
