8. Quantum Numbers

Overview

Four quantum numbers: n = principal level, l = subshell (0 = s, 1 = p, 2 = d, 3 = f), ml = orbital orientation, ms = spin (+½ or −½).

Maximum electrons per level = 2n2.

Pattern: Level → Shape → Position → Spin (memory hook: “L‑S‑P‑S”).

Practice Questions

  1. 1. Which quantum number identifies the energy level?

    n

  2. 2. What values can l take for the d-subshell?

    l = 2

  3. 3. How many electrons can a level hold according to the 2n² rule when n = 3?

    18

  4. 4. What quantum number represents electron spin?

    m_s

  5. 5. What is the name of the orbital orientation quantum number?

    m_l

Early-Buzz Patterns

  • "This quantum number specifies energy level…" → n
  • "This quantum number describes subshell shape…" → l
  • "This rule limits electrons to 2n²…" → Maximum electron cap
  • "This quantum number takes values of ±½…" → Spin quantum number
  • "This quantum number specifies orbital orientation…" → m_l
