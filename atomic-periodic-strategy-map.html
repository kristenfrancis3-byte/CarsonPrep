Quick Reference · Strategy Map
Atomic & Periodic Strategy Map
1. Identify Question Type (First 3 Words)
"This particle…" → Subatomic
"This rule…" → Electron Configuration
"This trend…" → Periodic Trends
"This equation…" → EM/Nuclear Equations
"As ___ increases…" → Zeff & Shielding
▼
2. Snap to Category Instantly
The first recognizable keyword locks the category.

Examples:
- "subatomic" → Proton/Neutron/Electron
- "rule states" → Aufbau / Hund / Pauli
- "weighted average" → Atomic Mass
- "minimum frequency" → Threshold Frequency
▼
3. Trigger Mapping (Instant Answer)
Opposite spins → Pauli
Singly before pairing → Hund
Lowest energy orbitals → Aufbau
Positive ion smaller → Cation
Negative ion larger → Anion
Weighted average → Atomic Mass
Relates wavelength/frequency → c = λν
▼
4. Predict the Ending (Before It's Said)
KAAC clues are short. Once the pattern appears, the rest is predictable.

Examples:
- "This rule states that electrons…" → Aufbau
- "This trend increases down…" → Atomic Radius
- "This effect requires threshold…" → Photoelectric Effect
- "As Zeff increases…" → Radius decreases
▼
5. Auto-Answer by Pattern Recognition
Carson should confidently buzz before the final clue.

State-Champ Reflexes:
- "same electrons…" → Isoelectronic
- "mass difference…" → Mass Defect
- "minimum frequency…" → Threshold Frequency
- "energy required to remove…" → Ionization Energy
▼
6. High-ROI Knowledge Clusters
These generate ~70% of KAAC questions in this category:

✔ Periodic Trend Directions
✔ Aufbau / Hund / Pauli
✔ c = λν and E = hν
✔ Cation vs Anion size rules
✔ Zeff & Shielding logic
