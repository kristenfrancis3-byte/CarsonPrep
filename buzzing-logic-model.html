Buzzing Logic Model

Here is the process state champions follow mentally — in order — for this category.

Step 1 — Identify the QUESTION TYPE

In the first 1–3 words:

  • "This particle…" → Subatomic
  • "This rule…" → Configurations
  • "This trend…" → Periodic trends
  • "This equation…" → EM or nuclear
  • "As X increases…" → Trend or Zeff

This tells Carson exactly which part of the brain to activate.

Step 2 — Identify the KEYWORD

Every question in this category hinges on one clue word:

  • "opposite spins" → Pauli
  • "singly before pairing" → Hund
  • "lowest energy orbitals" → Aufbau
  • "weighted average" → atomic mass
  • "percent abundance" → isotopes
  • "positive ion" → cation
  • "larger radius ion" → anion
  • "mass difference" → mass defect

Carson should memorize these as instant triggers.

Step 3 — Predict the ending

KAAC questions are short. If the first half matches a known template, the second half is predictable.

  • "This rule states that electrons…" → fill lowest → Aufbau
  • "This trend increases down…" → radius
  • "This equation relates wavelength…" → c = λν
  • "This subatomic particle that differs…" → neutron
  • "This effect requires threshold…" → photoelectric
  • "Increasing shielding causes…" → larger radius

If Carson sees the pattern, he should buzz before the question is done.

Step 4 — Answer with Confidence

Because he's pattern-buzzing, the key is to trust the structure, not the missing part. Carson's competitive advantage is pattern recognition → not waiting for confirmation.
