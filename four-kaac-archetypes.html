The Four KAAC Question Archetypes

KAAC science questions in this category fall into four predictable "archetype buckets." Carson should identify the type immediately when the question begins.

1. Definition-in-Disguise Questions

Start with:

  • "This rule…"
  • "This trend…"
  • "This subatomic particle…"
  • "This quantity…"
  • "This effect…"

These ALWAYS point to a single formula or concept.

Carson's Strategy:
When you hear a category keyword (rule, particle, trend, effect), instantly lock into the subcategory. Don't wait for the rest of the clue.

2. Relationship/Trend Questions

Start with:

  • "As ___ increases…"
  • "Greater ___ leads to…"
  • "A higher number of protons causes…"
  • "This value decreases because…"

These test cause-and-effect thinking.

Carson's Strategy:
Immediately ask:
"Is this about Zeff or shielding?"
90% of trend questions reduce to those two words.

3. Equation/Identity Questions

Start with:

  • "This equation relates…"
  • "This constant equals…"
  • "The formula for photon energy…"
  • "E equals h times…"

These appear in the EM and nuclear subsections.

Carson's Strategy:
Identify the formula type quickly:
  • Wavelength–frequency → c = λν
  • Energy–frequency → E = hν
  • Mass–energy → E = mc²
Buzz as soon as it's clear which formula is being described.

4. Comparison/Ranking Questions

Start with:

  • "Which is larger…"
  • "Which has higher…"
  • "Rank these ions…"
  • "Which has lower ionization energy…"

These test periodic trends or ion sizes.

Carson's Strategy:
Use the UP-RIGHT rule instantly:
  • Size increases down-left
  • Everything else increases up-right
No need to think — it becomes reflexive.
