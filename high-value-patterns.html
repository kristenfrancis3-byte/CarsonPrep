High-Value Patterns KAAC Writers Repeat

These are extremely common across years.

Pattern 1: "This rule…"

Answer is always one of three:

  • Aufbau
  • Hund
  • Pauli

Buzz after the first 3–5 words.

Pattern 2: "Weighted average…"

This always means atomic mass. Buzz instantly.

Pattern 3: "Increases down, decreases across…"

Answer = atomic radius, unless the clue mentions metallic character.

Pattern 4: "Energy required to remove…"

Answer = ionization energy.

Pattern 5: "As wavelength decreases…"

Answer = energy increases or frequency increases.

Pattern 6: "Positive ions have…"

Answer = smaller radius (cations shrink).

Pattern 7: "Same electrons but different sizes…"

Answer = isoelectronic series (then rank by proton count).

Pattern 8: "minimum frequency required…"

Answer = threshold frequency.

Pattern 9: "mass difference between predicted…"

Answer = mass defect.

Pattern 10: "most stable nuclei…"

Answer = iron.
