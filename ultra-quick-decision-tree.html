Ultra-Quick Decision Tree

What Carson Actually Uses in Competition

As soon as the question starts, match the opening words to an instant answer pattern:

  • "This particle…" → Proton / Electron / Neutron
  • "This rule…" → Aufbau / Hund / Pauli
  • "Weighted average…" → Atomic mass
  • "Same element…" → Isotopes
  • "As X increases…" → Zeff or shielding
  • "This trend…" → Radius / Ionization / Electronegativity
  • "This equation…" → c = λν / E = hν / E = mc²
  • "Positive ions…" → Smaller
  • "Negative ions…" → Larger
  • "Same electrons…" → Isoelectronic
  • "Minimum frequency…" → Threshold
  • "Mass difference…" → Mass defect

This is exactly how state champions get so fast.

Memorization Strategy

Carson should practice these patterns until they become automatic. The goal is to lock onto the pattern in the first 1–3 words and buzz with confidence.

Train with flashcards, quizzes, and timed drills. When these mappings are reflexive, Carson will have the speed advantage needed to dominate this category.
