4. Electron Configuration Rules

Overview

Aufbau Principle: Electrons fill lowest energy levels first.

Pauli Exclusion Principle: Each orbital holds 2 electrons with opposite spins.

Hund’s Rule: Electrons half‑fill orbitals before pairing.

Pattern: Aufbau → Pauli → Hund (A → P → H).

Shortcut: Noble gas shorthand, e.g., Na = [Ne]3s1.

Magnetism: Paramagnetic = unpaired electrons; Diamagnetic = paired.

Practice Questions

  1. 1. What rule states that electrons fill the lowest-energy orbitals first?

    Aufbau principle

  2. 2. What rule requires electrons in the same orbital to have opposite spins?

    Pauli exclusion principle

  3. 3. What rule states that electrons fill orbitals singly before pairing?

    Hund’s rule

  4. 4. If an element has unpaired electrons, is it paramagnetic or diamagnetic?

    Paramagnetic

  5. 5. Write the noble-gas shorthand configuration for magnesium.

    [Ne]3s²
